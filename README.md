# Локальный бот документов

## Запуск
1) Создайте `.env` рядом с `README.md` и добавьте:
   `BOT_TOKEN=...`
2) Установите зависимости:
   `pip install -r requirements.txt`
3) Запустите бота:
   `python -m src.main`

## Отчетность в группе (темы)
1) Включите "Темы" (форум) в группе, добавьте бота и дайте ему право писать сообщения.
2) Откройте тему "Запуски" и отправьте команду: `/set_topic_starts`.
3) Откройте тему "Файлы" и отправьте команду: `/set_topic_files`.
4) Проверка: напишите боту `/start` в ЛС и сформируйте документ — лог должен прийти в "Запуски", PDF уйдет в "Файлы".

## Документы и шаблоны
- Шаблоны находятся в `src/documents/templates/`.
- Для PDF на Windows требуется установленный Microsoft Word (используется `docx2pdf`).
- При ошибке конвертации бот уведомит и отправит DOCX как fallback.
